在MIPS汇编语言中，数据声明用于在程序的数据段（`.data`）中定义变量、数组、字符串等静态数据，其核心格式为`名称: 存储类型 值` 。

### **一、基本声明格式**
```mips
变量名: .存储类型 初始值(或大小)
```
- **变量名**：标识符，后跟冒号（`:`）。
- **存储类型**：指定数据类型，如 `.word`、`.byte`、`.space` 等。
- **初始值/大小**：根据类型赋初值或分配空间。

### **二、常见存储类型及示例**
1. **`.word`**  
   定义32位整数（4字节），可赋初值或留空。  
   ```mips
   var1: .word 3       # 定义整数变量var1，初值为3
   var2: .word         # 定义变量var2，未初始化
   ```

2. **`.byte`**  
   定义8位字符或字节，可赋多个初值。  
   ```mips
   char1: .byte 'A'    # 定义字符变量char1，值为'A'
   array: .byte 'a', 'b', 'c'  # 定义3字节数组，初始化为'a','b','c'
   ```

3. **`.space`**  
   分配未初始化的连续内存空间（字节数）。  
   ```mips
   buffer: .space 40   # 分配40字节未初始化空间
   ```

4. **`.asciiz`**  
   定义以空字符（`\0`）结尾的字符串。  
   ```mips
   msg: .asciiz "Hello"  # 定义字符串"Hello"，末尾自动加\0
   ```

5. **`.half`**  
   定义16位半字（2字节），较少使用。  
   ```mips
   half_var: .half 1000  # 定义半字变量，值为1000
   ```

### **三、声明位置与程序结构**
MIPS汇编程序通常分为两部分：
1. **数据段（`.data`）**  
   声明所有静态数据，位于代码开头。  
   ```mips
   .data
   var1: .word 10
   str: .asciiz "MIPS"
   ```

2. **代码段（`.text`）**  
   包含程序指令，以 `main:` 为入口。  
   ```mips
   .text
   main:
       lw $t0, var1    # 加载var1的值到寄存器$t0
       li $v0, 10       # 系统调用号10（退出程序）
       syscall
   ```

### **四、完整示例**
```mips
.data
    num: .word 42          # 32位整数
    chars: .byte 'X', 'Y' # 2字节数组
    buffer: .space 16     # 16字节未初始化空间
    text: .asciiz "MIPS"  # 字符串

.text
main:
    lw $t0, num           # 加载num到$t0
    li $v0, 10            # 退出程序
    syscall
```

### **五、注意事项**
1. **数据对齐**：MIPS要求字（4字节）地址为4的倍数，半字（2字节）地址为2的倍数。
2. **未初始化数据**：`.space` 或未赋初值的变量内容不确定，需谨慎使用。
3. **字符串处理**：`.asciiz` 自动添加终止符，适合系统调用输出。
